# 🥒 Pickle

A modern, premium e-commerce website for an authentic Nepali pickle brand. Built with **React**, **Tailwind CSS**, and **Framer Motion**, the project focuses on storytelling, premium product presentation, responsive layouts, and smooth user interactions.

> Designed and developed from research to implementation.

---

## ✨ Features

- 🎨 Modern & minimal UI
- 📱 Fully responsive design
- ⚡ Smooth animations with Framer Motion
- 🧩 Reusable component architecture
- 🎯 Design token based color system
- 🛍️ Product showcase
- 💬 Customer testimonials carousel
- 🚀 Optimized performance
- ♿ Accessible and scalable layout

---

## 🛠️ Tech Stack

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Vite


---

## 🎨 Design Philosophy

The website is designed around a premium food branding experience.

Some design principles followed:

- Consistent typography hierarchy
- Spacious layout
- Minimal color palette
- Visual storytelling
- Smooth micro-interactions
- Reusable UI components
- Scalable design token system
